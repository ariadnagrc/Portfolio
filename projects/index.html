DBledus

Programa diseñado para medir y visualizar el nivel de voz del usuario en tiempo real, proporcionando un indicador claro y fácil de interpretar en una escala de 0 a 100.

Este programa destaca por su originalidad al integrar bibliotecas de audio en un entorno Java para capturar y analizar el nivel de sonido mediante el micrófono del sistema. A pesar de que el tratamiento de audio en Java no es tan habitual como en otros lenguajes más orientados al multimedia, he sabido adaptarme rápidamente a las particularidades de las librerías disponibles, superando desafíos técnicos como la gestión en tiempo real de las entradas del micrófono, la normalización de los datos de audio y la interpretación eficiente de los niveles sonoros. Este enfoque demuestra no solo mi capacidad para trabajar con tecnologías poco convencionales, sino también mi facilidad para aprender y aplicar nuevas bibliotecas según lo requiera el proyecto.

Data Alchemist

Herramienta diseñada para el proyecto Eramus+ BIDUSA. Su objetivo fue agilizar el proceso de conversión de datos de un formato a otro, eliminando esa parte manual.

Este proyecto es una herramienta hecha en Python que convierte automáticamente archivos CSV o Excel a formato ARFF, el cual es muy utilizado en entornos de análisis de datos como Weka. Además de la conversión, el programa incluye una función de previsualización gráfica que permite al usuario visualizar los datos antes de exportarlos, facilitando la comprensión y revisión de la información. Es una solución práctica para quienes trabajan con machine learning y necesitan transformar y revisar datasets de forma rápida y sencilla, todo desde una misma aplicación.
